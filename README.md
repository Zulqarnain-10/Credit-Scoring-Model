# Credit Scoring Model

*A machine-learning model that predicts applicant creditworthiness from financial and demographic data.*

## Problem

Lenders need to assess default risk before extending credit. A data-driven model scores applicants quickly and consistently, supporting faster and fairer decisions.

## Approach

Cleaned and preprocessed historical financial data with feature engineering for key risk signals, then trained and compared classification models in scikit-learn, evaluating with standard classification metrics.

## Tech Stack

`Python` · `scikit-learn` · `pandas` · `NumPy` · `Jupyter`

## How to Run

Install dependencies (`pip install pandas numpy scikit-learn`), then open `Credit_Scoring_Model.ipynb` in Jupyter and run all cells.
